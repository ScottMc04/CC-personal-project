Mountain Club Website

I incorporated responsive design by using media queries. I also included the :hover CSS Selector to make the nav bar more interactive. I practiced my newly learnt skills of CSS Flexbox which was used to create the layout I desired.


Luxe Laundry Project
This project was created in python. It automates the process of taking in linen to be cleaned. It does this by the use of functions, loops, if statements, file management and more...

This project was created closly following client requirements. The program starts by adding a custom order which is saved to a .txt file and once the user is ready they get a choice to 'view' orders, 'update' orders or cancel their orders. Next a search algorithm is used to find the users order and performs said task. Any completed order is added to the archive .txt file with relevant information like 'Order Name', 'Amount paid' and 'Towels Washed'.

pAeqour project
This JavaScript project was made for a research team that had found a new mysterious organism. I was tasked to create objects that simulate the DNA of the P. aeqour organism for the research team.

Multiple functions were used, one of them comparing the DNA of 2 simulated pAeqour. Objects and methods were created to store these new organisms. There is also a method to calculate the survival chance of the organisms with certain DNA structures.
The random DNA bases were created using the 'Math' built-in object and for loops to cycle through each DNA base individually.
